EXPLANATION:

If "is_master_pool_rewards_method" == true:
    a. bonding to be done for the master pool,
    b. Consider child pool ids and distribute accordingly
        i. use ["*"] if all pools are needed else specify pool_ids
        ii. rewards will be provided for the mentioned pool ids if liquidity is provided (without bonding) 
    c. "lock_duration" will be ignored on provided child pool ids.
    d. but liquidity to be provided in child pools 24hours before epoch start to get rewards,
Else:
    a. The rewards will act as in external rewards.
    b. child pool ids will be ignored
    c. rewards will be distributed for this pool only,
    d. "lock_duration" will be considered i.e it will be checked if the liquidity is bonded or not.
